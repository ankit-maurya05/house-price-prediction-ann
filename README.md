# house-price-prediction-ann
This project demonstrates how to build and evaluate an ANN regression model using Keras. The workflow includes preprocessing, handling outliers, log transformation, early stopping to prevent overfitting, and detailed evaluation using sklearn metrics.

## Overview
This project implements an Artificial Neural Network (ANN) for a regression problem using Keras and TensorFlow.

## Dataset
- Number of features: 9
- Target variable: Continuous
